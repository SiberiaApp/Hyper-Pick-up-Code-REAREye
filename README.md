# Hyper Pick-up Code 澎湃记 背屏组件

> 适配澎湃记的小米背屏组件，基于 REAREye 驱动

## 安装前提

1. 下载[澎湃记](https://github.com/badnng/Hyper-pick-up-code)
2. 下载[REAREye最新测试版本](https://updates.uwu.hk/)

---

## 安装方法

### 一、自动安装

在 REAREye 组件商店安装

### 二、手动安装

1. 下载组件
2. 导入组件至**组件模板管理器**，组件名称可任意填写
3. 进入 **Scene路由管理器**，填写以下内容：
   * **目标包名**：`com.Badnng.moe`
   * **Scene**：`CH:promoted_live_update_channel`
   * **组件名称**：上面填写的组件名称
4. 重启背屏
